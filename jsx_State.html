Code Challenge: Constructing Lists


In the JS section of this Codepen is a React class component called App that is rendered to the view. For the following challenges, you will only need to edit code in the JS section, not the HTML or CSS.

  • Challenge 1: Add a fourth country to the list stored in App's state.
  • Challenge 2: Cause the continent to show up next to each country name in the view, as in: "Laos - Asia"
  • Challenge 3: There is a CSS class called "title" defined in the CSS section. Apply this class to the h1 element for "My Travel Bucket List".

NOTE: When React talks about "lists and keys", it does not necessarily mean an HTML listitem element (<li>). It could be a list composed from <span> elements, <div> elements, et cetera, typically rendered from multiple items in an array which are all treated to the same markup. In this Codepen example, <div> is being used.

Code Challenge: Passing Props


In the JS section of this Codepen is a React class component called App that contains some information about an upcoming party in its state. For the following challenges, you will only need to edit code in the JS section, not the HTML or CSS.

  • Challenge 1: Between the div tags at lines 18 and 20 of the JS section, render a component named <Party> and pass the partyType, entertainment, and venue properties from the state.
  • Challenge 2: Create/define a new React component named Party. In it, have it return a div that contains h3 tags surrounding the words Party Time!, followed by two paragraph elements that use the passed-in props. The first paragraph element will say: This _____ party will be held at ______ with a maximum capacity of ____. The second paragraph element will say: Featured entertainment: ______! The blanks will be filled in using props. See the screenshot in the learning portal page for this code challenge.
  • Challenge 3: Add another property to the state in App, pass it as props to the Party component, and use it somewhere in the Party component. The property is up to you. Some examples: an array with a list of guest names, a cake flavor, the name of the birthday person, the address of the venue, etc!
