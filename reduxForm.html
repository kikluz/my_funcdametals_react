React Redux Form

Install react-redux-form and convert the controlled form in the Contact component to a form whose state is kept in the Redux store.

Instructions NOTE: At the end of this exercise, when you start your project, you will see warning messages that say Imported JSX component text must be in PascalCase or SCREAMING_SNAKE_CASE Please disregard these messages. They will not affect your code. First, install react-redux-form library in the project folder, using Yarn:

yarn add react-redux-form@1.16.14 Then open ContactComponent.js and update the feedback form to use react-redux-form: . . . import { Breadcrumb, BreadcrumbItem, Button, Label, Col, Row } from 'reactstrap'; . . . import { Control, LocalForm } from 'react-redux-form'; . . . handleSubmit(values) { console.log("Current state is: " + JSON.stringify(values)); alert("Current state is: " + JSON.stringify(values)); } . . . this.handleSubmit(values)}>
. . . You will also need to remove the code related to validation that you added in the Controlled Form Validation exercise. Refer to the video for this exercise for what part of the code to remove. Save all changes and test. NOTE: When you start your project, you will see warning messages that say Imported JSX component text must be in PascalCase or SCREAMING_SNAKE_CASE Please disregard these messages. They will not affect your code. Optional: Do a Git commit with the message "React Redux Form". Summary In this lesson, you installed the react-redux-form library and used its LocalForm and Control components to convert the form in the Contact component to use the Redux store as its single source of truth for form state.
