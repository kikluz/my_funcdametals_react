Learn to use the JavaScript ternary operator.

Instructions

The ternary operator (also known as the conditional operator) is so-called because it's the only JavaScript operator with three operands. (Ternary means "composed of three parts".) condition to be evaluated ? expression to execute if condition is truthy : expression to execute if condition is falsy

It can often be used as a shortcut for the if statement. Open your browser's developer console and enter the following: 


const lightswitch = {
    switchState: "on",
    flipSwitch() {
        if (this.switchState === "on") {
            this.switchState = "off";
        } else {
            this.switchState = "on";
        }
    }
};

This creates a lightswitch object with a switchState property initialized to "on", along with a flipSwitch method that toggles the switchState property to its opposite. Enter the following into your console: 


lightswitch.flipSwitch();
lightswitch

lightswitch

This should show you that the lightswitch's switchState property is now flipped to "off". If you call the lightswitch.flipSwitch() method again, it should flip the switchState back to "on".
Next, overwrite the flipSwitch method in the lightswitch object by entering the following into your console: 


lightswitch.flipSwitch = function() {
    (this.switchState === "on") ? this.switchState = "off" : this.switchState = "on";
};

Note that the set of parentheses around the condition to be evaluated is optional, but can be useful for readability.

Now call the lightswitch.flipSwitch() method again and check the contents of the lightswitch object again afterward: 


lightswitch.flipSwitch();

lightswitch

You should see that the flipSwitch() method is working the same as before. For another example, enter this code into your console: 


function canVote(age) {
    if (age >= 18) {
        return true;
    }
    return false;
}

This function takes an age argument and returns true if it's 18 or over, and false otherwise. You might think that you can rewrite this with the ternary operator like so: 


function canVote(age) {
    (age >= 18) ? return true : return false;
}

But if you tried this, it would cause an error. The return statement cannot be used inside a ternary operation because it's not an expression, it's a statement. In order to mimic the original function, you can return the result of the ternary operation, like so: 


function canVote(age) {
    return (age >= 18) ? true: false;
}

A word of advice: While the ternary operator can be useful and simplify your code, one drawback is that it can also compromise the readability (and the ease of debugging) your code. Consider this before using it every time; for example, a particularly complex section of code may benefit from the use of a clear if/else statement rather than the ternary operator.

Summary

In this lesson, you learned two different ways to use the ternary operator as a shorthand for if statements -- on its own, and as a return value.
