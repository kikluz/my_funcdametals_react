React Transitions

Learn a way to implement animated transitions in your React project. Instructions We will use a third-party library called react-transition-group to animate transitions between views. Install the library in your project folder as follows, using yarn add: yarn add react-transition-group@4.4.1 Then open App.css and add the following classes: . . . .page-enter { opacity: 0; } .page-enter-active { opacity: 1; transition: 300ms ease-out; } .page-exit { opacity: 1; } .page-exit-active { opacity: 0; transition: 300ms ease-in; } Next, open MainComponent.js and add in the following to configure the animated transition: . . . import { TransitionGroup, CSSTransition } from 'react-transition-group'; . . .
} /> } /> } />
. . . Test your app in the browser. You should notice a subtle fade effect when switching between views now. Don't forget to start json-server first! Next, for another demonstration of using CSS transitions, update App.css once more: . . . .page-enter { opacity: 0; transform: translateX(-100%); } .page-enter-active { opacity: 1; transform: translateX(0%); transition: 300ms ease-out; } . . . Save all the changes. Optional: Do a Git commit with the message "React Transitions". Summary In this lesson, you learned how to implement an animated transition for when the route switches to different views, using the react-transition-group library.
