Week 3 Workshop Assignment

Objectives

In the React application you have been developing, you will add a new module named AboutComponent.js that will be responsible for the About Us view of your single-page application. By the end of this assignment, you should have completed the following tasks:

Integrated the provided AboutComponent.js file into the single page application. Added a new functional component named RenderPartner to AboutComponent.js that renders the details of a given partner using the Reactstrap Media component.
Construct and render the list of partners in the About Us page using the RenderPartner component.
