Exercise: React Redux Form Validation

Learn to implement simple form validation for controlled forms using react-redux-form. Instructions Implement simple form validation Open ContactComponent.js and update it as follows to implement form validation: . . . import { Control, LocalForm, Errors } from 'react-redux-form'; const required = val => val && val.length; const maxLength = len => val => !val || (val.length <= len); const minLength = len => val => val && (val.length >= len); const isNumber = val => !isNaN(+val); const validEmail = val => /^[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,4}$/i.test(val); . . . . . . Save the changes. Optional: Do a Git commit with the message "React Redux Form Validation" Summary In this lesson, you learned to configure functions to validate form inputs, then use them with the Control and Errors components from react-redux-form to show error messages when validation has failed.
